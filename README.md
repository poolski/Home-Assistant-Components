# Home Assistant Components

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)

[![Open your Home Assistant instance and open a repository inside the Home Assistant Community Store.](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=poolski&repository=Home-Assistant-Components&category=integration)

A collection of custom Home Assistant integrations.

---

## Components

| Component | Description |
| --- | --- |
| [Statistics Outlier Cleaner](docs/statistics_outlier_cleaner.md) | Detect and fix outlier spikes in long-term statistics. Corrects both `sum` and `state` to prevent re-injection after HA restarts. Supports scheduled automation via service actions. |

---

## Installation

All components in this repo are available as a single HACS custom repository. Click the button above or follow the steps in each component's documentation.
